# Phishing Website Detection System

## Description
This project detects phishing websites using Machine Learning.
The system takes a URL as input and predicts whether it is safe or phishing.

## Technologies Used
- Python
- Flask
- Machine Learning
- Logistic Regression
- TF-IDF Vectorizer
- HTML & CSS

## How to Run
1. Install required libraries
2. Run `train_model.py`
3. Run `app.py`
4. Open browser and go to:
   http://127.0.0.1:5000/

## Output
- SAFE website
- PHISHING website
